Progress tracking lands in Kanchi, plus a tiny SDK for emitting events.

## What's new

**Task progress and steps**
Kanchi now ingests `kanchi-task-progress` and `kanchi-task-steps` events, stores
progress history, and renders a progress bar with step details in the task view.
A new `/api/tasks/{task_id}/progress` endpoint exposes the latest snapshot and
history. See the [Task progress guide](/docs/core/task-progress).

**Kanchi SDK (alpha)**
A new `kanchi-sdk` package provides typed helpers and a task mixin for emitting
progress and step events. It is optional; you can emit the same events directly
with Celery if you prefer.
