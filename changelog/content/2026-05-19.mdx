A small patch release to keep Docker deployments running cleanly.

## What's fixed

**Database drivers restored**
The Docker image now includes the required database drivers again. This fixes a
regression where containerized deployments could start without the drivers needed
to connect to the configured database.

Upgrade as usual: pull the latest image and restart your services.
